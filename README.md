# OpenAI ChatBot

Simple chatbot using OpenAI GPT-5 API.

## Setup

1. Clone the repository
2. Install dependencies:
   ```bash
   pip install openai python-dotenv
   ```
3. Create `.env` file:
   ```
   OPENAI_API_KEY=your_api_key_here
   ```
4. Run the chatbot:
   ```bash
   python chatbot.py
   ```

## Usage

- Type your message and press Enter
- Type "exit" to quit

## Requirements

- Python 3.6+
- OpenAI API key
- GPT-5 access
